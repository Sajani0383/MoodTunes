# 🎵 Mood Tunes – Java Music Recommendation App

A **Java-based console application** that suggests songs based on your mood.  
Users can create personal playlists, add favorites, and manage songs with JSON storage.

---

## ✨ Features
- 🎧 Mood-based song recommendations (`happy`, `sad`, `relaxed`, etc.)
- ❤️ Add or view favorite songs
- 📂 Create and manage playlists
- 🧑‍💻 Admin: add or remove songs dynamically
- 💾 Persistent data saved in JSON (using Gson)
- 👤 User personalization (name & email session)

---

## 🧰 Technologies Used
- **Java 17+**
- **Gson Library** (for JSON read/write)
- **OOP Concepts** – Classes, Objects, Encapsulation, File Handling, Collections
